# Watch Face Editor :: WFE
[![](https://jitpack.io/v/hms-douglas/wfe.svg)](https://jitpack.io/#hms-douglas/wfe)

A library for building Wear OS watch faces editors/customizers, that uses the Jetpack Watch Face API!
